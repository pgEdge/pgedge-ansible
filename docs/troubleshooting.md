# Troubleshooting

This guide provides solutions to common issues encountered when deploying
and managing pgEdge clusters with the Ansible Collection. Select a topic
below to find solutions for a specific component.

- [Collection Installation](troubleshooting/installation.md) - Issues that
  block cluster deployment before it begins.
- [Package Repository](troubleshooting/repository.md) - Issues that prevent
  software installation.
- [System Configuration](troubleshooting/system.md) - Host-level
  configuration issues.
- [Postgres](troubleshooting/postgres.md) - Database initialization and
  startup issues.
- [etcd](troubleshooting/etcd.md) - Issues with the distributed
  coordination layer.
- [Patroni](troubleshooting/patroni.md) - HA controller issues.
- [HAProxy](troubleshooting/haproxy.md) - Proxy routing issues.
- [Spock Replication](troubleshooting/spock.md) - Multi-zone replication
  issues.
- [PgBackRest](troubleshooting/backup.md) - Backup and recovery issues.
